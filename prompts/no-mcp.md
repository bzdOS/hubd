# hubd — any surface WITHOUT MCP (web chats, anything)

Paste when you want to capture a dialog (this is the Harvest Protocol's
graceful degradation):

```
Extract from this dialog: (1) projects touched — slug + 3-6 line digest;
(2) action items — [project] action / owner / due / importance, implicit ones too;
(3) decisions with the why; (4) role changes; (5) open questions.
Then output ONE shell code block containing only ready-to-paste `hub` CLI
commands (hub card / hub task add / hub report), properly quoted. Nothing else.
```

Paste the block into any terminal where hubd is installed. Done — the next
agent's hub_brief knows everything.
